# Gavion

**AI-Powered Solutions for Dental Practitioners**

Gavion helps dental practices modernize their operations with intelligent automation tools.

## Services

- **AI Websites** — Custom, conversion-optimized websites built for dental practices
- **AI Chatbot** — 24/7 patient-facing chatbot for appointment booking, FAQs, and triage
- **AI Receptionist** — Automated phone and messaging assistant that handles scheduling, reminders, and follow-ups
- **Patient Engagement** — Smart recall systems, review management, and personalized outreach
